# go-lddb

Package **lddb** provides tools for working with the **LD-DB** (**linked-data database**) format, for the Go programming language.

This can store data from **Linked-Data** formats such as:
**CBOR-LD**,
**JSON-LD**,
etc.

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-lddb

[![GoDoc](https://godoc.org/github.com/reiver/go-lddb?status.svg)](https://godoc.org/github.com/reiver/go-lddb)

## Import

To import package **lddb** use `import` code like the following:
```
import "github.com/reiver/go-lddb"
```

## Installation

To install package **lddb** do the following:
```
GOPROXY=direct go get github.com/reiver/go-lddb
```
